  信息和每个人相关，**信息素养实践课程可以帮助同学们更好地利用网络信息**。在信息技术如此发达的今天，互联网给我们提供了广阔而又丰富的信息资源空间。信息素养实践课程其实是交给我们一种方法，**一种更快捷方便、更高效获取信息的方法**，让我对计算机技术解决实际问题的能力有了更多期待，让我更为主动地去了解更多解决问题可以的程序、软件，去主动了解更多获取信息的渠道。*主动去获取信息，才能适应信息时代的变化*。
